Write a program in Java with an algorithm to display a student mark sheet with total marks, percentage, and grade.

import java.util.Scanner;

public class StudentMarksheet {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Step 1: Input student details
        System.out.print("Enter Student Name: ");
        String name = sc.nextLine();

        System.out.print("Enter Roll Number: ");
        int rollNo = sc.nextInt();

        // Step 2: Input marks for 5 subjects
        int subjects = 5;
        int[] marks = new int[subjects];
        int total = 0;

        for (int i = 0; i < subjects; i++) {
            System.out.print("Enter marks for Subject " + (i + 1) + ": ");
            marks[i] = sc.nextInt();
            total += marks[i];
        }

        // Step 3: Calculate percentage
        double percentage = (double) total / subjects;

        // Step 4: Determine grade
        char grade;
        if (percentage >= 90) grade = 'A';
        else if (percentage >= 80) grade = 'B';
        else if (percentage >= 70) grade = 'C';
        else if (percentage >= 60) grade = 'D';
        else if (percentage >= 50) grade = 'E';
        else grade = 'F';

        // Step 5: Display Marksheet
        System.out.println("\n----- Student Mark Sheet -----");
        System.out.println("Name: " + name);
        System.out.println("Roll No: " + rollNo);
        for (int i = 0; i < subjects; i++) {
            System.out.println("Subject " + (i + 1) + ": " + marks[i]);
        }
        System.out.println("Total Marks: " + total);
        System.out.printf("Percentage: %.2f%%\n", percentage);
        System.out.println("Grade: " + grade);

        sc.close();
    }
}

Diamond Pattern 

                import java.util.Scanner;

public class DiamondPattern {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Ask user for number of rows for upper half
        System.out.print("Enter number of rows for the upper half: ");
        int n = sc.nextInt();

        // Upper half (including middle row)
        for (int i = 1; i <= n; i++) {
            // Print spaces
            for (int j = i; j < n; j++) {
                System.out.print(" ");
            }
            // Print stars
            for (int k = 1; k <= (2 * i - 1); k++) {
                System.out.print("*");
            }
            System.out.println();
        }

        // Lower half
        for (int i = n - 1; i >= 1; i--) {
            // Print spaces
            for (int j = n; j > i; j--) {
                System.out.print(" ");
            }
            // Print stars
            for (int k = 1; k <= (2 * i - 1); k++) {
                System.out.print("*");
            }
            System.out.println();
        }

        sc.close();
    }
}

Write a program in Java with an algorithm to print transpose of a matrix. 

                import java.util.Scanner;

public class MatrixTranspose {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Step 1: Input number of rows and columns
        System.out.print("Enter number of rows: ");
        int rows = sc.nextInt();
        System.out.print("Enter number of columns: ");
        int cols = sc.nextInt();

        // Step 2: Declare and input the matrix
        int[][] matrix = new int[rows][cols];
        System.out.println("Enter matrix elements:");
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                System.out.print("Element [" + i + "][" + j + "]: ");
                matrix[i][j] = sc.nextInt();
            }
        }

        // Step 3: Compute transpose
        int[][] transpose = new int[cols][rows];
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                transpose[j][i] = matrix[i][j];
            }
        }

        // Step 4: Print the transpose
        System.out.println("\nTranspose of the matrix:");
        for (int i = 0; i < cols; i++) {
            for (int j = 0; j < rows; j++) {
                System.out.print(transpose[i][j] + " ");
            }
            System.out.println();
        }

        sc.close();
    }
}

Write a java program with an algorithm that implements a multi-thread application that has three threads. First thread generates random integer every 1 second and if the value is even, second thread computes the square of the number and prints. If the value is odd, the third thread will print the value of cube of the number. 

                import java.util.Random;

class NumberHolder {
    private Integer number = null;
    private boolean numberAvailable = false;

    // Synchronized method to set the number
    public synchronized void setNumber(int num) throws InterruptedException {
        while (numberAvailable) {
            wait();  // wait if previous number not processed
        }
        number = num;
        numberAvailable = true;
        notifyAll();
    }

    // Synchronized method to get the number
    public synchronized int getNumber() throws InterruptedException {
        while (!numberAvailable) {
            wait();  // wait if no number available
        }
        numberAvailable = false;
        notifyAll();
        return number;
    }
}

class NumberGenerator extends Thread {
    private NumberHolder holder;
    private Random random = new Random();

    public NumberGenerator(NumberHolder holder) {
        this.holder = holder;
    }

    @Override
    public void run() {
        try {
            while (true) {
                int num = random.nextInt(100);  // Random integer between 0 and 99
                System.out.println("Generated number: " + num);
                holder.setNumber(num);
                Thread.sleep(1000);  // wait 1 second before generating next number
            }
        } catch (InterruptedException e) {
            System.out.println("NumberGenerator interrupted");
        }
    }
}

class EvenSquareCalculator extends Thread {
    private NumberHolder holder;

    public EvenSquareCalculator(NumberHolder holder) {
        this.holder = holder;
    }

    @Override
    public void run() {
        try {
            while (true) {
                int num = holder.getNumber();
                if (num % 2 == 0) {
                    System.out.println("Square of " + num + " is " + (num * num));
                } else {
                    // If odd, put it back and let odd thread handle it
                    holder.setNumber(num);
                    Thread.sleep(10);  // slight delay to prevent busy waiting
                }
            }
        } catch (InterruptedException e) {
            System.out.println("EvenSquareCalculator interrupted");
        }
    }
}

class OddCubeCalculator extends Thread {
    private NumberHolder holder;

    public OddCubeCalculator(NumberHolder holder) {
        this.holder = holder;
    }

    @Override
    public void run() {
        try {
            while (true) {
                int num = holder.getNumber();
                if (num % 2 != 0) {
                    System.out.println("Cube of " + num + " is " + (num * num * num));
                } else {
                    // If even, put it back and let even thread handle it
                    holder.setNumber(num);
                    Thread.sleep(10);  // slight delay to prevent busy waiting
                }
            }
        } catch (InterruptedException e) {
            System.out.println("OddCubeCalculator interrupted");
        }
    }
}

public class MultiThreadApp {
    public static void main(String[] args) {
        NumberHolder holder = new NumberHolder();

        Thread generator = new NumberGenerator(holder);
        Thread evenThread = new EvenSquareCalculator(holder);
        Thread oddThread = new OddCubeCalculator(holder);

        generator.start();
        evenThread.start();
        oddThread.start();
    }
}

Write a Java program with an algorithm to insert the specified element at the specified position in the linked list. 

                class Node {
    int data;
    Node next;

    public Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class LinkedList {
    Node head;

    // Insert element at specified position
    public void insertAtPosition(int data, int position) {
        Node newNode = new Node(data);

        if (position < 0) {
            System.out.println("Invalid position. Must be >= 0.");
            return;
        }

        if (position == 0) {
            // Insert at the head
            newNode.next = head;
            head = newNode;
            return;
        }

        Node current = head;
        int count = 0;

        // Traverse to the node before the desired position
        while (current != null && count < position - 1) {
            current = current.next;
            count++;
        }

        if (current == null) {
            System.out.println("Position is beyond the length of the list.");
            return;
        }

        // Insert the new node
        newNode.next = current.next;
        current.next = newNode;
    }

    // Print the linked list
    public void printList() {
        Node current = head;
        while (current != null) {
            System.out.print(current.data + (current.next != null ? " -> " : ""));
            current = current.next;
        }
        System.out.println();
    }

    public static void main(String[] args) {
        LinkedList list = new LinkedList();

        // Initial list: 10 -> 20 -> 30
        list.insertAtPosition(10, 0);
        list.insertAtPosition(20, 1);
        list.insertAtPosition(30, 2);
        System.out.print("Initial List: ");
        list.printList();

        // Insert 15 at position 1: 10 -> 15 -> 20 -> 30
        list.insertAtPosition(15, 1);
        System.out.print("After inserting 15 at position 1: ");
        list.printList();

        // Insert 5 at position 0: 5 -> 10 -> 15 -> 20 -> 30
        list.insertAtPosition(5, 0);
        System.out.print("After inserting 5 at position 0: ");
        list.printList();

        // Insert 35 at position 5 (end): 5 -> 10 -> 15 -> 20 -> 30 -> 35
        list.insertAtPosition(35, 5);
        System.out.print("After inserting 35 at position 5: ");
        list.printList();

        // Attempt invalid insertion
        list.insertAtPosition(40, 10);
    }
}

Write a Java program with an algorithm for Student Information Management System with Database Cnnectivity. 


                CREATE DATABASE school_db;
USE school_db;

CREATE TABLE students (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    age INT,
    course VARCHAR(100)
);


import java.sql.*;
import java.util.Scanner;

public class StudentManagementSystem {

    // Database connection parameters
    private static final String JDBC_URL = "jdbc:mysql://localhost:3306/school_db";
    private static final String JDBC_USERNAME = "root";
    private static final String JDBC_PASSWORD = "your_password";

    private Connection connection;

    public StudentManagementSystem() throws SQLException {
        // Establish DB connection
        connection = DriverManager.getConnection(JDBC_URL, JDBC_USERNAME, JDBC_PASSWORD);
    }

    // Add a student
    public void addStudent(String name, int age, String course) throws SQLException {
        String sql = "INSERT INTO students (name, age, course) VALUES (?, ?, ?)";
        try (PreparedStatement stmt = connection.prepareStatement(sql)) {
            stmt.setString(1, name);
            stmt.setInt(2, age);
            stmt.setString(3, course);
            int rows = stmt.executeUpdate();
            System.out.println(rows + " student(s) added.");
        }
    }

    // View all students
    public void viewStudents() throws SQLException {
        String sql = "SELECT * FROM students";
        try (Statement stmt = connection.createStatement();
             ResultSet rs = stmt.executeQuery(sql)) {

            System.out.println("ID | Name | Age | Course");
            System.out.println("-----------------------------");

            while (rs.next()) {
                int id = rs.getInt("id");
                String name = rs.getString("name");
                int age = rs.getInt("age");
                String course = rs.getString("course");

                System.out.printf("%d | %s | %d | %s%n", id, name, age, course);
            }
        }
    }

    // Update a student
    public void updateStudent(int id, String name, int age, String course) throws SQLException {
        String sql = "UPDATE students SET name = ?, age = ?, course = ? WHERE id = ?";
        try (PreparedStatement stmt = connection.prepareStatement(sql)) {
            stmt.setString(1, name);
            stmt.setInt(2, age);
            stmt.setString(3, course);
            stmt.setInt(4, id);
            int rows = stmt.executeUpdate();
            System.out.println(rows + " student(s) updated.");
        }
    }

    // Delete a student
    public void deleteStudent(int id) throws SQLException {
        String sql = "DELETE FROM students WHERE id = ?";
        try (PreparedStatement stmt = connection.prepareStatement(sql)) {
            stmt.setInt(1, id);
            int rows = stmt.executeUpdate();
            System.out.println(rows + " student(s) deleted.");
        }
    }

    public void close() throws SQLException {
        if (connection != null) connection.close();
    }

    public static void main(String[] args) {
        try (Scanner scanner = new Scanner(System.in)) {
            StudentManagementSystem sms = new StudentManagementSystem();
            boolean running = true;

            while (running) {
                System.out.println("\n--- Student Information Management System ---");
                System.out.println("1. Add Student");
                System.out.println("2. View Students");
                System.out.println("3. Update Student");
                System.out.println("4. Delete Student");
                System.out.println("5. Exit");
                System.out.print("Enter choice: ");

                int choice = scanner.nextInt();
                scanner.nextLine();  // consume newline

                switch (choice) {
                    case 1:
                        System.out.print("Enter name: ");
                        String name = scanner.nextLine();
                        System.out.print("Enter age: ");
                        int age = scanner.nextInt();
                        scanner.nextLine(); // consume newline
                        System.out.print("Enter course: ");
                        String course = scanner.nextLine();

                        sms.addStudent(name, age, course);
                        break;

                    case 2:
                        sms.viewStudents();
                        break;

                    case 3:
                        System.out.print("Enter student ID to update: ");
                        int updateId = scanner.nextInt();
                        scanner.nextLine();
                        System.out.print("Enter new name: ");
                        String newName = scanner.nextLine();
                        System.out.print("Enter new age: ");
                        int newAge = scanner.nextInt();
                        scanner.nextLine();
                        System.out.print("Enter new course: ");
                        String newCourse = scanner.nextLine();

                        sms.updateStudent(updateId, newName, newAge, newCourse);
                        break;

                    case 4:
                        System.out.print("Enter student ID to delete: ");
                        int deleteId = scanner.nextInt();
                        scanner.nextLine();

                        sms.deleteStudent(deleteId);
                        break;

                    case 5:
                        running = false;
                        sms.close();
                        System.out.println("Exiting system...");
                        break;

                    default:
                        System.out.println("Invalid choice. Try again.");
                }
            }
        } catch (SQLException e) {
            System.out.println("Database error: " + e.getMessage());
        }
    }
}
